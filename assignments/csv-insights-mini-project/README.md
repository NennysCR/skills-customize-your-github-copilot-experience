# 📘 Assignment: CSV Insights Mini-Project

## 🎯 Objective

Practice basic data analysis with Python by loading a CSV file, computing simple summaries, and describing one clear insight from the data.

## 📝 Tasks

### 🛠️ Load and Explore the Dataset

#### Description
Open the provided CSV file and print the first few rows and key summary statistics.

#### Requirements
Completed program should:

- Read the CSV file using Python
- Print the first 5 rows of the dataset
- Print the average, minimum, and maximum values for a numeric column

### 🛠️ Find a Trend and Explain It

#### Description
Use the dataset to identify one interesting pattern and write a short explanation.

#### Requirements
Completed program should:

- Calculate one simple comparison or summary (for example, highest score or average by group)
- Print the result clearly
- Write 1–2 sentences explaining what the result means
